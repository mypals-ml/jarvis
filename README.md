# JARVIS - Local Home-Hosted AI Assistant

JARVIS is a private, local, and home-hosted AI assistant designed for personal and family use. It provides advanced intelligence and conversational abilities without relying on external cloud providers.

## Key Features
*   **Conversational Intelligence:** High-quality text and voice chat.
*   **Smart Home Mastery:** Direct control of home devices via Home Assistant.
*   **Knowledge & Memory:** Internet search capabilities and long-term context retention.
